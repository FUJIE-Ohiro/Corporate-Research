# 企业研究模板 — Company Research Template

> 本模板定义了每家公司研究的标准文件结构和每个模块需要覆盖的关键内容。

## 文件结构

每家公司在 `companies/<company_name>/` 下包含以下文件：

| 文件 | 内容 |
|------|------|
| `00_snapshot.md` | 一页总览：是否值得申请、匹配度、最终建议 |
| `01_company_overview.md` | 公司做什么、历史、产品、客户、全球定位 |
| `02_business_model.md` | 赚钱逻辑、增长引擎、护城河、弱点 |
| `03_industry_and_competition.md` | 行业空间、竞争格局、技术路线、强弱分析 |
| `04_financials.md` | 营收/利润/现金流（上市）或规模/趋势（非上市） |
| `05_newgrad_recruiting.md` | 新卒招聘全貌：岗位、要求、流程、薪资、难度 |
| `06_work_style_and_career.md` | 企业文化、WLB、晋升、海外机会、跳槽出口 |
| `07_fit_for_me.md` | 我的匹配分析：优势、短板、ES/面试策略 |
| `08_risks_and_questions.md` | 行业/公司/招聘/个人风险 + 待验证问题 |
| `09_xiaohongshu_summary.md` | 小红书图文脚本（15-20页） |
| `sources.md` | 所有信息来源分类整理 |

## 每个模块关注点

### 00_snapshot
- 公司一句话介绍
- 是否值得求职申请
- 最适合申请的岗位
- 适合什么类型的人
- 最大优点 / 最大风险
- 我的匹配度判断
- 最终建议：强推 / 可投 / 观望 / 不优先

### 01_company_overview
- 核心产品/业务、历史沿革
- 日本国内定位 / 全球定位
- 主要客户和应用场景
- 上下游关系

### 02_business_model
- 怎么赚钱、哪些业务最赚钱
- 成长驱动、护城河
- 未来天花板、商业模式弱点
- 对应届生最有发展前景的方向

### 03_industry_and_competition
- 行业空间与增长驱动
- 日本国内 + 全球竞争格局
- 技术路线差异
- 这家公司强在哪、弱在哪

### 04_financials
- **上市公司**：营收、利润率、现金流、资本开支、近3-5年变化、管理层表态
- **非上市公司**：公开营收规模、母公司支持、行业景气、招聘扩张节奏、新投资动向
- 严格区分事实与推断

### 05_newgrad_recruiting（核心模块）
- 招聘对象、职种、留学生可否、文理要求
- 专业/日语/英语要求
- 工作地点、年收/福利
- 选考流程与难度
- 是否适合我

### 06_work_style_and_career
- 加班/WLB、晋升路径、轮岗
- 海外机会、技术积累
- 未来跳槽出口

### 07_fit_for_me
- 我适不适合 / 该申什么岗
- 简历如何对标、面试故事
- 综合评分（兴趣/胜率/成长/薪资/长期发展）

### 08_risks_and_questions
- 行业/公司/招聘/个人风险
- 需要继续验证的问题清单

### 09_xiaohongshu_summary
- 15-20页内容脚本
- 每页：标题 + 一句主结论 + 3-5要点 + 配图建议
- 附：小红书标题、钩子、互动引导、标签

### sources
- 按类别整理：官网、招聘、IR、新闻、口碑、行业报告
- 标注证据薄弱处
