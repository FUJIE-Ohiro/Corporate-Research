# 05 新卒招聘 — Keyence / キーエンス

> ⭐ 本模块是整个研究的**核心**。Keyence 的新卒招聘竞争极其激烈，但对理工科修士来说，回报也是天花板级别的。

## 结论
Keyence 面向全学历、全学科招聘新卒，理系修士在商品開発等技术岗上有显著优势。留学生可以申请，但必须具备商务级日语。选考难度极高（倍率推定70-200倍），以"论理性思考力"和"行動力"为核心评价轴。通过后，回报是日本最高级别的薪资和成长环境。

## 选考流程

Keyence 的选考以"**不公开求める人物像**"著称——他们故意不告诉你想要什么样的人，要看你"ありのまま"的表现。

### 典型流程（推定，每年可能微调）

```
1. エントリー（マイページ登録）
   ↓
2. 説明会 / セミナー参加
   ↓
3. ES提出（Webまたは手書き / フェイスシート形式）
   ↓
4. 適性検査（能力検査 + 独自性格検査 ※複数回）
   ↓
5. 一次面接（個人）
   ↓
6. 二次面接（深掘り）
   ↓
7. 最終面接
   ↓
8. 内定
```

### 选考の特徴

| 特征 | 说明 |
|------|------|
| **適性検査が超重要** | 独自の性格検査が複数回実施され、非常に重視されている |
| **論理的思考力** | 全ての面接で"なぜ？なぜ？"の深掘りが来る |
| **行動力・主体性** | 過去の経験で"自ら動いた"エピソードが問われる |
| **ストレス耐性** | 圧迫面接ではないが、テンポの速い質疑応答で対応力を見る |
| **一貫性** | 性格検査と面接回答の一貫性を照合している（推定） |
