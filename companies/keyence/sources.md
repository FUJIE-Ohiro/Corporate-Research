| 结论 | 置信度 | 説明 |

| 選考倍率 70-200倍 | 中 | 公式非公開。複数の就活情報サイトの推定値を採用 |
| 年収推移（1年目700万→3年目1000万等） | 中 | OB口コミ・転職サイト情報に基づく推定。個人差大 |

| 学歴フィルターの存在 | 中 | 公式には否定するが、体感的に存在するとの口コミ多数 |
| クラス制の具体的年収レンジ | 中 | 口コミベース。公式情報なし |
| 商品開発の大阪配属 | 中-高 | 本社が大阪のため蓋然性高いが、東京R&D拠点の可能性もゼロではない |
| 創始者退任後の経営力 | 低 | 長期的な観察が必要。現時点では業績好調のため問題なし |
| 留学生の実際の採用人数/比率 | 低 | 公開情報なし。OB訪問等で要確認 |
