# 02 商业模式 — Keyence / キーエンス

## 结论
Keyence 的商业模式是**"ファブレス（无工厂）× 直販（直销）× 高付加価値"**三位一体的组合。它通过不持有工厂压低固定成本、通过直销抓住客户需求并实现高定价、通过持续推出"世界初"的产品维持技术壁垒。这一模式让它成为全球制造业中营业利润率最高的企业之一。

---

## 公司怎么赚钱

```
[客户工厂的课题] → [Keyence 直販营业发现需求]
                         ↓
              [商品开发部门快速研发产品]
                         ↓
              [外部协力工厂生产]
                         ↓
              [Keyence 以高附加值价格直接销售给客户]
```

**赚钱公式**：
- 高价值产品 × 不经过中间商 × 不持有重资产 = **超高利润率**
- 售价中的利润率远高于行业平均，因为客户愿意为"解决问题"付费

## 哪些业务是增长引擎

1. **画像処理/机器视觉** — AI时代制造业品质检查的核心，需求持续爆发
2. **3D测量/轮廓仪** — 半导体先进封装、EV电池等高精度场景增长
3. **海外市场** — 全球FA自动化渗透率仍在上升，特别是东南亚和印度

## 盈利模式的关键支柱

### 1. ファブレス（无工厂）
- 不持有生产设备 → 固定成本极低
- 轻资产模式 → 高 ROE、高现金流
- 将资源集中在**企划、开发、营业**这三个高价值环节

### 2. 直販（直销）
- 全部由自社営業直接销售，不经过代理商
- 好处1：**价格主导权** — 不被中间商压价
- 好处2：**需求洞察** — 营业在客户现场发现痛点 → 直接反馈给开发部门
- 好处3：**客户粘性** — 建立深度信赖关系

### 3. 高付加価値产品战略
- 新产品 70%+ 是"世界初/業界初"
- 追求"客户还没意识到自己需要"的产品（潜在ニーズ）
- 不打价格战，用产品力和解决方案力维持高定价

## 成本结构

| 成本类型 | 说明 |
|----------|------|
| 原材料/制造 | 委托外部，变动成本为主，弹性大 |
| 人工成本 | 高薪但员工数精简（"最小の人で最大の成果"） |
| 研发 | 持续投入但用高效的小团队模式 |
| 营销/渠道 | 直販模式下无代理商佣金，但需要维持庞大的直销团队 |
| 设备固定成本 | 极低（ファブレス） |

**关键指标**：营业利润率约 **52%**（FY2025.3），意味着每赚100円留下52円利润。这在日本制造业中遥遥领先。

## 商业模式的弱点

1. **对直販团队的重度依赖** — 如果营业力下降，核心竞争力直接受损
2. **ファブレス的风险** — 供应链中断时自身没有生产备选（但目前管理良好）
3. **单一事业结构** — 所有鸡蛋放在FA这一个篮子里（但FA覆盖行业非常广）
4. **高薪模式的可持续性** — 如果利润率下降，薪资体系可能面临调整压力
5. **中国市场的不确定性** — 地缘政治和经济下行可能影响重要的中国市场收入
