## 使用方式

在 markdown 中引用：
```markdown

```

在小红书脚本中，这些图可直接对应以下页面：
- 第3页（商业模式）→ `business_model_flow.png`
- 第4页（薪资）→ `salary_progression.png`
- 第9页（选考流程）→ `selection_process.png`
- 第12页（跳槽出口）→ `career_exit_paths.png`
- 第13页（竞争对手）→ `competitor_comparison.png`
