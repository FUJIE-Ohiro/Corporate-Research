# Sources (信息来源)

本研究涉及的 Kioxia (キオクシア) 相关数据基于以下公开来源（更新至2024年底/2026新卒披露期）：

## 1. 官方披露资料
- **Kioxia Holdings Corporation 决算短信/Earnings Release**
  - FY2023 财报 (显示巨亏数据)
  - FY2024 Q1/Q2 财报 (显示单季最高利润与营收复苏数据)
  - 链接: `https://www.kioxia-holdings.com/ja-jp/ir/library/earnings.html`
- **Kioxia 新卒採用公式サイト (Recruit Page)**
  - 核心来源：2026年募集要項、募集职种与初任给暴涨数据（364,900円/月）
  - 链接: `https://www.kioxia.com/ja-jp/about/recruit/new-grads.html`

## 2. 行业与新闻报道
- **日经跨国科技报道 (Nikkei Asia / 日本経済新聞)**
  - 检索词：`キオクシア 業績`，`キオクシア ウエスタンデジタル 統合`
  - 内容：关于 Kioxia 与西部数据 (WD) 合并流局、政府补贴以及四日市/北上新厂房稼动的情况。

## 3. 员工口碑评价 (口コミ)
- **OpenWork / 就活会議 / 外資就活**
  - 获取了关于“WLB”、“纵向组织架构”、“借上社宅超级划算”以及“四日市生活真相”的内部员工评价。
  - 数据指出其实际总年收增长存在一定的滞后性和部门差异性。

## 4. 特别说明
- Kioxia 目前仍为**未上市**状态（截至本报告撰写时）。部分长期平均年收数据是以控股公司为基准或者依靠口碑网站推算的，实际个体（特别是首几年）的收入浮动极大取决于其公司业绩分红的波动性。
